# AQA Test Task - Inforce

## Setup and Run Tests

### Prerequisites

- Node.js (LTS version)
- npm

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/kelkrif/aqa-test-task-inforce.git
   cd aqa-test-task-inforce

2. Install dependencies:
    ```bash
    npm install

3. Running Tests

Execute the tests with:
   ```bash
    npx playwright test-main:ui
